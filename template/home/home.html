Welcome

Welcome to the Shadow Frameworks. Our goal is to provide a basis for you to build your own creations on. It provides a core template with several files to help you focus on what is important, your code. There is a skeleton admin section provided at {{full_url}}admin (username is admin, password adminadmin). All views inherit the controller class which provides the tools to easily pass information to your views (stored in template). The model class provides easy access to the database, allowing you to focus more on manipulating and analyzing data over writing querys. And the best thing is, it is all written to help you organize code into a very modular MVC system so that you can reuse/change different elements with little to no hassle. I hope you enjoy the frameworks, If you end up using and liking it please credit me.
